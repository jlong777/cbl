INTRO_LIBCBL(3B)                                   Last changed: 03-09-04

NAME

	INTRO_LIBCBL - Introduction to the Portable Cray Bioinformatics
                       Library routines.
	       
IMPLEMENTATION

	Version 1.0 for 32/64-bit big/little endian unix platforms.

DESCRIPTION

	LIBCBL routines perform low level bit manipulation and
	searching operations useful in the analysis of nucleotide
	and amino acid sequence data.

	Data Compression routines:

		cb_compress
		cb_uncompress

	Reverse Complement routine:

		cb_revcompl

	Nucleotide to Amino Acid Translation routine:

		cb_amino_translate_ascii

	Sequence Characterization routine:

		cb_countn_ascii

	Search routines:
	
		cb_searchn
		cb_repeatn
		cb_swa_fw
		cb_swa_fw_align
		cb_swa_fw_init
		cb_sw_fw_score

	File Format routines:

		cb_read_fasta
		cb_fasta_convert
	
	Bit routines:

		cb_copy_bits
		cb_irand

	Misc routines:

		cb_free
		cb_malloc
		cb_version

NOTES

	This implementation is the portable version of the original
	proprietary CRAY CBL. It was written at the Arctic Region
	Supercomputing Center (ARSC), University of Alaska Fairbanks.
